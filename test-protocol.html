Troubleshooting

  • Nothing happens when I click: Make sure you've installed Sorcery Desktop to /Applications (run make install)
  • Browser shows error: Check that the protocol is registered: defaults read /Applications/Sorcery\ Desktop.app/Contents/Info CFBundleURLTypes
  • Wrong app opens: Re-register: make install-quick
  • Permission denied: Allow the browser to open external apps in browser settings
